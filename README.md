# AR Balloon Shooting (Unity AR Demo)

## üéÆ Project Overview  
**AR Balloon Shooting** is a mobile augmented-reality demo built with Unity and AR Foundation.  
Players aim and shoot floating balloons in their real-world environment using their device camera. The game features balloon spawning, raycast-based shooting mechanics, pop effects, and a scoring system.

## ‚úÖ Features  
- AR camera (mobile) integration for real-world overlay  
- Balloons float upward and can be interacted with via touch/aim  
- Particle smoke effect when a balloon is hit  
- Scoring system with simple UI  
- Lightweight and optimized for mobile platforms  

## üì± AR Details  
- Platform: Android (via APK)  
- Requirements: AR-capable mobile device  
- Interaction: Tap or aim + shoot mechanism using the AR camera forward vector  
- Purpose: Demonstrates core AR mechanics in a simple, clean environment  

## üõ† Technologies

- AR Foundation / ARCore / ARKit (as supported)  
- C# scripting

## üöÄ Getting Started  
1. Clone the repository or download the ZIP.  
2. Open the folder in Unity Hub (select the `AR Balloon Shooting` root directory).  
3. Ensure AR Foundation, AR Core (or ARKit) packages are installed.  
4. Connect an AR-capable mobile device, switch build target to Android or iOS, and build & run.  
5. Or simply download the APK via the itch.io link above to test directly on your phone.
